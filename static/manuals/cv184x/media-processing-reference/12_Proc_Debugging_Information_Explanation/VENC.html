【调试信息分析】

记录当前视频编码属性配置以及状态信息。

【参数说明】

参数

描述

MODULE PARAM

VencBufferCache

编码的码流 buffer 是否使用 cache 模式

0: no

1: yes

FrameBufRecycle

用于存储参考帧和高级smartP帧的空闲缓冲区在编码期间是否被回收

0: 不回收

1: 回收

VencMaxChnNum

编码通道的最大数量

VENC CHN ATTR1

ID

编码通道 ID

Width

编码通道宽度

Height

编码通道高度

Type

编码通道类型

ByFrame

获取码流的方式

0: packet 方式

1: frame 方式

Sequence

序号

当按帧获取流时，它代表帧序列号。

当按包获取流时，它代表包序列号。

LeftBytes

流缓冲区中的剩余字节

LeftFrm

流缓冲区中剩余帧的数量

CurPacks

当前帧的包数量

(当前无效)

GopMode

GOP 模式

prio

通道优先级

VENC CHN ATTR 2

VeStr

是否开始编码

SrcFr

VENC用于控制帧速率的源帧速率(输入帧速率)

TarFr

VENC用于控制帧速率的目标帧速率

Timeref

繁忙队列中最新帧的 Timeref

PixFmt

正在编码的帧的格式

PicAddr

正在编码的帧的地址

WakeUpFrmCnt

当通道使用超时或以块模式获取流时，指定唤醒块接口的帧数

VENCCROP INFO

ID

编码通道 ID

CropEn

是否启用VENC频道的裁剪功能

StartX

要裁剪的图像的起始水平坐标

StartY

要裁剪的图像的起始垂直坐标

Width

裁剪图像的宽度

Height

裁剪图像的高度

ROI INFO

ID

通道号

Index

感兴趣区域(ROI)索引

bRoiEn

是否启用 ROI

bAbsQp

ROI是否使用绝对QP模式

Qp

由 ROI 配置的 QP 值

Width

ROI 宽度 (unit: pixel)

Height

ROI 高度 (unit: pixel)

StartX

ROI的起始水平坐标 (unit: pixel)

StartY

ROI的起始垂直坐标 (unit: pixel)
VENC PTS
STATE
通道接收到的帧的时间戳

ID

通道号

RcvFirstFrmPts

通道收到的第一个帧的时间戳

RcvFrmPts

通道接收的当前帧的时间戳

VENC CHN PERFORMANCE

ID

通道号

InFPS

编码通道的输入 FPS (from VI or VPSS or app)

OutFPS

编码通道的输出 FPS

HwEncTime

硬件编码时间成本

CVITEK DEBUG STATE

VencDebugMask

middleware 的调试掩码

VencStartFrmIdx

middleware 调试的开始帧

VencEndFrmIdx

middleware 调试的结束帧

VencDumpPath

YUV src帧转储路径
