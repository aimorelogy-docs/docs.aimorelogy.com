【参数说明】

参数

描述
REGION STATUS OF OVERLAY
OVERLAY状态

Hdl

OVERLAY的Handle号。

Type

OVERLAY类型，值为0。

Used

该资源是否被占用。

N：未占用。

Y：占用。

PiFmt

OVERLAY像素格式， 参看PIXEL_FORMAT_E。

W

OVERLAY宽度（单位：像素）。

H

OVERLAY高度（单位：像素）。

Color

OVERLAY颜色。

Phy

OVERLAY占用内存的物理地址。

Virt

OVERLAY占用内存的虚拟地址。

Stride

OVERLAY内存跨度（单位：byte）。

CnvsNum

OVERLAY内存数目。

Cmpr

画布是否压缩的。

N：未压缩的。

Y：压缩的。

MaxNeedIon

OVERLAY最大需要内存。
