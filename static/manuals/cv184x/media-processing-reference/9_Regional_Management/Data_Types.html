9.4. 数据类型¶

RGN 模块相关数据类型定义如下：

  • RGN_MIN_WIDTH ：定义区域的最小宽度。

  • RGN_MIN_HEIGHT ：定义区域的最小高度。

  • RGN_COVER_MAX_WIDTH ：定义 COVER 区域的最大宽度。

  • RGN_COVER_MAX_HEIGHT ：定义 COVER 区域的最大高度。

  • RGN_COVER_MIN_X ：定义 COVER 区域起始位置的 X 坐标最小值。

  • RGN_COVER_MIN_Y ：定义 COVER 区域起始位置的 Y 坐标最小值。

  • RGN_COVER_MAX_X ：定义 COVER 区域起始位置的 X 坐标最大值。

  • RGN_COVER_MAX_Y ：定义 COVER 区域起始位置的 Y 坐标最大值。

  • RGN_COVEREX_MAX_NUM ：定义 COVEREX 区域的最大个数。

  • RGN_COVEREX_MAX_WIDTH ：定义 COVEREX 区域的最大宽度。

  • RGN_COVEREX_MAX_HEIGHT ：定义 COVEREX 区域的最大高度。

  • RGN_COVEREX_MIN_X ：定义 COVEREX 区域起始位置的 X 坐标最小值。

  • RGN_COVEREX_MIN_Y ：定义 COVEREX 区域起始位置的 Y 坐标最小值。

  • RGN_COVEREX_MAX_X ：定义 COVEREX 区域起始位置的 X 坐标最大值。

  • RGN_COVEREX_MAX_Y ：定义 COVEREX 区域起始位置的 Y 坐标最大值。

  • RGN_OVERLAY_MAX_WIDTH ：定义 OVERLAY 区域的最大宽度。

  • RGN_OVERLAY_MAX_HEIGHT ：定义 OVERLAY 区域的最大高度。

  • RGN_OVERLAY_MIN_X ：定义 OVERLAY 区域起始位置的 X 坐标最小值。

  • RGN_OVERLAY_MIN_Y ：定义 OVERLAY 区域起始位置的 Y 坐标最小值。

  • RGN_OVERLAY_MAX_X ：定义 OVERLAY 区域起始位置的 X 坐标最大值。

  • RGN_OVERLAY_MAX_Y ：定义 OVERLAY 区域起始位置的 Y 坐标最大值。

  • RGN_MOSAIC_MAX_NUM ：定义 MOSAIC 区域的最大个数。

  • RGN_MOSAIC_X_ALIGN ：定义 MOSAIC 区域的 X 坐标对齐。

  • RGN_MOSAIC_Y_ALIGN ：定义 MOSAIC 区域的 Y 坐标对齐。

  • RGN_MOSAIC_WIDTH_ALIGN ：定义 MOSAIC 区域的宽度对齐。

  • RGN_MOSAIC_HEIGHT_ALIGN ：定义 MOSAIC 区域的高度对齐。

  • RGN_MOSAIC_MIN_WIDTH ：定义 MOSAIC 区域的最小宽度。

  • RGN_MOSAIC_MIN_HEIGHT ：定义 MOSAIC 区域的最小高度。

  • RGN_MOSAIC_MAX_WIDTH ：定义 MOSAIC 区域的最大宽度。

  • RGN_MOSAIC_MAX_HEIGHT ：定义 MOSAIC 区域的最大高度。

  • RGN_MOSAIC_MIN_X ：定义 MOSAIC 区域起始位置的 X 坐标最小值。

  • RGN_MOSAIC_MIN_Y ：定义 MOSAIC 区域起始位置的 Y 坐标最小值。

  • RGN_MOSAIC_MAX_X ：定义 MOSAIC 区域起始位置的 X 坐标最大值。

  • RGN_MOSAIC_MAX_Y ：定义 MOSAIC 区域起始位置的 Y 坐标最大值。

  • RGN_MAX_LAYER_VPSS ：定义一个 VPSS 通道最多可显示的区域层数。

  • RGN_ODEC_LAYER_VPSS ：定义一个 VPSS 通道的 ODEC 区域层。

  • RGN_NORMAL_LAYER_VPSS ：定义一个 VPSS 通道的 NORMAL 区域层。

  • RGN_MAX_NUM_VPSS ：定义一个 VPSS 通道最多可显示的区域的个数。

  • RGN_EX_MAX_NUM_VPSS ：定义扩展区域的最大个数。

  • RGN_EX_MAX_WIDTH ：定义扩展区域的最大宽度。

  • RGN_MAX_BUF_NUM ：定义区域使用的最大内存数量。

  • RGN_MAX_NUM ：定义区域的最大个数。

  • RGN_INVALID_HANDLE ：定义区域无效句柄。

  • RGN_CMPR_MIN_SIZE ：定义区域压缩的最小尺寸。

  • RGN_HANDLE ：定义区域句柄。

  • RGN_TYPE_E ：定义区域类型。

  • RGN_AREA_TYPE_E ：定义 COVER、COVEREX 区域类型。

  • OSD_COMPRESS_MODE_E ：定义 OSD 压缩模式类型。

  • OSD_COMPRESS_INFO_S ：定义 OSD 压缩模式属性。

  • COVER_CHN_ATTR_S ：定义 COVER 区域的通道属性。

  • COVEREX_CHN_ATTR_S ：定义 COVEREX 区域的通道属性。

  • OVERLAY_ATTR_S ：定义视频迭加区域的属性。

  • OVERLAY_CHN_ATTR_S ：定义视频迭加区域的通道属性。

  • OVERLAYEX_ATTR_S ：定义扩展视频迭加区域的属性。

  • OVERLAYEX_CHN_ATTR_S ：定义扩展视频迭加区域的通道属性。

  • RGN_ATTR_U ：定义区域属性联合体。

  • RGN_CHN_ATTR_U ：定义区域通道属性联合体。

  • RGN_ATTR_S ：定义区域属性结构体。

  • RGN_CHN_ATTR_S ：定义区域通道属性结构体。

  • RGN_COORDINATE_E ：定义坐标类型。

  • RGN_QUADRANGLE_S ：定义任意四边形结构体。

  • MOSAIC_BLK_SIZE_E ：定义 MOSAIC 区域的块大小类型。

  • MOSAIC_CHN_ATTR_S ：定义 MOSAIC 区域通道属性。

  • INVERT_COLOR_MODE_E ：定义 OSD 反色模式。

  • OVERLAY_INVERT_COLOR_S ：定义 OSD 反色属性。

  • RGN_CMPR_TYPE_E ：定义区域压缩类型。

  • RGN_LINE_ATTR_S ：定义线条属性。

  • RGN_RECT_ATTR_S ：定义矩形属性。

  • RGN_BITMAP_ATTR_S ：定义位图属性。

  • RGN_CMPR_OBJ_ATTR_S ：定义压缩对象属性。

  • RGN_CANVAS_CMPR_ATTR_S ：定义画布压缩属性。

  • RGN_CANVAS_INFO_S ：定义画布信息。

  • RGN_RGBQUARD_S ：定义调色盘颜色。

  • RGN_COLOR_FMT_E ：定义 OSD 支持的颜色格式。

  • RGN_PALETTE_S ：定义调色盘属性。

typedef struct _OVERLAY_ATTR_S {

  CVI_BOOL bFlip;
  CVI_BOOL bMirror;
  OSD_COMPRESS_INFO_S stCompressInfo;
} OVERLAY_ATTR_S;

【成员】

成员名称

描述

enPixelFormat
像素格式。
PIXEL_FORMAT_ARGB_1555,
PIXEL_FORMAT_ARGB_4444,
PIXEL_FORMAT_ARGB_8888。

u32BgColor
区域背景色。
根据enPixelFormat定义。

stSize

区域宽高。

u32CanvasNum

区域内存数量。

bFlip
OSD内容是否翻转。
仅支持DUAL OS。

bMirror
OSD内容是否镜像。
仅支持DUAL OS。

stCompressInfo

OSD压缩模式属性。

【注意事项】

  • stSize会影响区域分配的内存大小，建议不要大于最终链接通道的宽高，以免浪费内存。

  • u32CanvasNum根据使用情况决定。
    若使用 CVI_RGN_SetBitMap，为1即可。
    若使用 CVI_RGN_GetCanvasInfo，希望double buffer避免更新过程中的瞬时，则建议为2。

【相关数据类型及接口】

无。
