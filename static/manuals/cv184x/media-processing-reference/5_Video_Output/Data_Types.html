5.4. 数据类型¶

  • VO_DEV:定义视频输出设备号。

  • VO_LAYER:定义视频输出设备视频层号。

  • VO_CHN ：定义通道号。

  • VO_INTF_TYPE_E:定义视频输出设备的接口类型。

  • VO_INTF_SYNC_E:定义视频输出设备的标准时序。

  • VO_LVDS_MODE_E : 定义LVDS接口模式。

  • VO_LVDS_OUT_BIT_E : 定义LVDS接口输出位宽。

  • VO_LVDS_LANE_ID : 定义LVDS接口的lane ID。

  • VO_CSC_MATRIX_E : 定义CSC转换矩阵。

  • VO_MAC_BT_MUX_E : 定义BT接口复用信号类型。

  • VO_MAC_D_SEL_E : 定义视频输出引脚选择枚举。

  • VO_BT_DATA_SEQ_E : 定义BT接口数据序列类型。

  • VO_BT_MODE_E : 定义BT接口模式枚举。

  • VO_CSC_S : 定义图像输出效果结构体。

  • VO_SYNC_INFO_S:定义视频输出设备的自定义时序结构体。

  • VO_LVDS_ATTR_S : 定义LVDS接口属性结构体。

  • VO_PUB_ATTR_S:定义视频输出设备公共属性结构体。

  • VO_VIDEO_LAYER_ATTR_S:定义视频输出设备视频层属性结构体。

  • VO_CHN_ATTR_S:定义视频输出设备通道属性结构体。

  • VO_MIN_CHN_WIDTH:定义视频输出设备通道最小宽度。

  • VO_MIN_CHN_HEIGHT:定义视频输出设备通道最小高度。

  • VO_MAX_DEV_NUM:定义视频输出设备最大设备数。

  • VO_MAX_LAYER_NUM:定义视频输出设备最大视频层数。

  • VO_MAX_CHN_NUM:定义视频输出设备最大通道数。

  • VO_I80_INSTR_S : 定义I80指令。

  • VO_PM_OPS_S : 电源管理函数指针结构体。

  • VO_GAMMA_INFO_S ：定义伽马校正信息。

  • VO_BIN_INFO_S : Gamma信息结构体。

  • VO_PATTERN_MODE : 定义显示图例模式。

  • VO_D_REMAP_S : 定义引脚映射结构体。

  • VO_PINMUX_S : 定义引脚PINMUX结构体。

  • VO_BT_ATTR_S : 定义BT接口引脚的配置结构体。

  • VO_MCU_INSTRS_S : 定义MCU接口初始化结构体。

  • VO_HW_MCU_CFG_S : 定义MCU接口的配置结构体。

  • VO_CHN_STATUS_S : 定义通道状态结构体。

  • VO_QUERY_STATUS_S ：定义视频输出通道状态结构体。
