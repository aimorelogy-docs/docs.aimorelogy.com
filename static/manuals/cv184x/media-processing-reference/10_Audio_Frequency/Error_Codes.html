10.5. 错误码¶

10.5.1. 音频基础属性错误码¶

Cvitek音频使用CVI_SUCCESS/CVI_FAILURE来代表基本的错误回传码:

#define CVI_SUCCESS 0

#define CVI_FAILURE (-1)

// 使用者需注意，CVI_TRUE/CVI_FALSE仅拿来做基本的判断回复，并未被使用作为成功或失败的判断:

#define CVI_TRUE 1

#define CVI_FALSE 0
