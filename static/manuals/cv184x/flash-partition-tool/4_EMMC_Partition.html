4. EMMC分区方式¶

4.1. 分区方式¶

分区方式是通过 Linux Kernel 的 cmdline partition 的方式进行分区管理而不是 elf partition；因此需要设置如下选项： 备注：默认已设置如下选项，用户不需要自行设置

CONFIG_BLK_CMDLINE_PARSER=y
CONFIG_PARTITION_ADVANCED=y
CONFIG_CMDLINE_PARTITION=y
CONFIG_EFI_PARTITION=n

详细可以参考下列信息: https://www.kernel.org/doc/Documentation/block/cmdline-partition.txt
