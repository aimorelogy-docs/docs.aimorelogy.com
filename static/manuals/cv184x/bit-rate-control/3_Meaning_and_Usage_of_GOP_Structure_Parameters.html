3. GOP结构参数意义和使用方法¶

3.1. 单参考P帧GOP结构属性说明及使用方法¶

单参考P帧编码GOP结构参数如下表所示。

参数

说明

使用建议

s32IPQpDelta

I帧与P帧的QP差值

调节I帧质量及控制I帧大小。正数代表I帧QP小于P帧QP, 设置适当差值可以减少呼吸效应

建议值：[2, 6]

3.2. 智能P帧GOP结构属性说明及使用方法¶

SmartP帧编码GOP结构参数如下表所示。

参数

说明

使用建议

u32BgInterval

长期参考帧间隔

需设置为Gop的整数倍，建议为10~12倍帧率

s32BgQpDelta

长期参考帧相对于普通P帧的QP差值

正数代表IDR帧QP小于P帧QP。

建议值：[4, 6]
