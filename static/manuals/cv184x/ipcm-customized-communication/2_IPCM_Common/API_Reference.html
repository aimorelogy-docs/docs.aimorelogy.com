2.3.9. CVI_IPCM_GetUserAddr¶

【描述】

返回对应物理地址的虚拟地址，物理地址的范围必须是alios管理的地址。

在 linux 侧，将从alios拿到的物理地址转成虚拟地址；

在 alios 侧，因为alios没有开mmc，所以直接返回alios的物理地址。

【语法】

CVI_VOID *CVI_IPCM_GetUserAddr(CVI_U32 u32Paddr);

【参数】

参数名称

描述

输入/输出

u32Paddr

待转换的物理地址

输入

【返回值】

返回值

描述

NULL

失败，物理地址不在alios管理的范围

其他

成功，返回对应物理地址的虚拟地址

【需求】

  • 头文件: cvi_type.h, cvi_ipcm.h

【注意】

无

【举例】

无

【相关主题】

无
