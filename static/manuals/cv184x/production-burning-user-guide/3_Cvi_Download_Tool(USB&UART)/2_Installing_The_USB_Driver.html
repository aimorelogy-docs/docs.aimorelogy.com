3.2. 安装usb驱动 (USB烧录)¶

以windows 10系统下驱动安装为例

步骤1

双击运行 CviUsbDownloadInstallDriver.exe ，点选下一步直至安装完成

驱动安装完成后， 即可使用 CviDownload 工具来烧录固件， 具体烧录步骤请参考《量产烧写使用指南》的 “cviDownloadTool 工具” 章节。

步骤2

找一片烧过 u-boot 的板子， 进入到uboot 后敲 cvi_utask vid 0x3346 pid 0x1000 ， 让cv18xx板子当作 device 接入电脑， 在电脑上打开“设备管理器”， 看到”libusb-win32 devices” 下面出现一个“CviUsbDownload” 的设备， 表示安装正常。

注解

不要重复安装驱动
