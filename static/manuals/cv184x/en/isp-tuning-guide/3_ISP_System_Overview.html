3. ISP System Overview¶

3.1. Function Overview¶

The ISP system supports standard image processing functions, including basic functions such as defective-pixel correction, lens shading correction, auto exposure, auto white balance, auto focus, and demosaicing. It also supports advanced processing functions such as noise reduction, WDR, and DRC.

The main image processing functions supported by the ISP are as follows:

  • Cropping (Crop)

  • Black Levelcorrection (BLC)

  • wide dynamic rangeFunction (WDR)

  • Static and dynamic defective-pixel correction (DPC)

  • Crosstalk removal (CrossTalk Removal)

  • Bayer noise reduction (BNR)

  • Lens shading correction (LSC)

  • Dynamic range compression (DRC)

  • Demosaicprocess

  • Purple-fringe correction (PFR)

  • Color correction (CCM)

  • Gammacorrection

  • Automatic dehazing (Dehaze)

  • 3D color lookup-table enhancement (CLUT)

  • Local contrast enhancement (LDCI)

  • Global contrast enhancement (DCI)

  • Image sharpening (Sharpen)

  • 3D noise reduction (3DNR)

  • Chroma noise processing (CNR)

  • Auto exposure (AE)

  • Auto focus (AF)

  • Auto white balance (AWB)

  • 3A-related statistics output

  • Brightness shading support
