2.4. eFuse API Reference¶

The eFuse API is in the EFUSE module and provides the following APIs:

  • CVI_EFUSE_GetSize: Query the size of an eFuse area.

  • CVI_EFUSE_Read: Read an eFuse area.

  • CVI_EFUSE_Write:Write an eFuse area.

  • CVI_EFUSE_EnableSecureBoot: Enable Secure Boot.

  • CVI_EFUSE_IsSecureBootEnabled: Query Secure Boot status.

  • CVI_EFUSE_EnableFastBoot: Enable Fast Boot.

  • CVI_EFUSE_IsFastBootEnabled: Query Fast Boot status.

  • CVI_EFUSE_Lock: Lock an eFuse area.

  • CVI_EFUSE_IsLocked: Query whether an eFuse area is locked.

  • CVI_EFUSE_LockWrite: Lock writes to an eFuse area.

  • CVI_EFUSE_IsWriteLocked: Query whether writes to an eFuse area are locked.
