2.3. eFuse U-Boot Command Reference¶

U-Boot provides the following commands to access eFuse:

  • efuser: Read an eFuse area.

  • efusew: Write an eFuse area.

2.3.1. efuser¶

[Description]

Read data from an eFuse area.

Syntax

efuser EFUSE_AREA

[Parameters]

Parameter Name

Description

EFUSE_AREA

eFuse area name, Reference eFuse User-Writable Areas and eFuse Security Configuration Fields.

[Example]

Read eFuse USER area data

cv184x# efuser USER
00000000: 03 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00    ................
00000010: 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00    ................
00000020: 00 00 00 00 00 00 00 00                            ........
cv184x#

2.3.2. efusew¶

[Description]

Write data to an eFuse area.

Syntax

efuser EFUSE_AREA DATA

[Parameters]

Parameter Name

Description

EFUSE_AREA

eFuse area name, Reference eFuse User-Writable Areas and eFuse Security Configuration Fields.

DATA

Data to write to eFuse, represented in hexadecimal
[Example]

Write the data 030201 to the user-defined area

cv184x# efusew USER 030201
Write eFuse USER(0) with:
00000000: 03 02 01                                           ...
cv184x# efuser USER
00000000: 03 02 01 00 00 00 00 00 00 00 00 00 00 00 00 00    ................
00000010: 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00    ................
00000020: 00 00 00 00 00 00 00 00                            ........
cv184x#
