4. U-boot Burning and Updating¶

4.1. Overview¶

U-boot Burning and UpdatingBurning and updating U-Boot requires burning the entire fip.bin (bootloader + U-Boot). The bootloader contains different DDR initialization parameters, which can be configured through the SDK menuconfig when selecting an EVB.
