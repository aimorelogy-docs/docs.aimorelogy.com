8.3. API Reference¶

This function module provides the following APIs to users:

  • CVI_VDEC_CreateChn: CreateVideo DecodingChannel.

  • CVI_VDEC_DestroyChn: DestroyVideo DecodingChannel.

  • CVI_VDEC_ResetChn: Reset the video decoding channel.

  • CVI_VDEC_GetChnAttr: GetVideo DecodingChannel attributes.

  • CVI_VDEC_SetChnAttr: SetVideo DecodingChannel attributes.

  • CVI_VDEC_StartRecvStream: DecodingdevicestartReceiveUserSendBitstream.

  • CVI_VDEC_StopRecvStream: DecodingdevicestopReceiveUserSendBitstream.

  • CVI_VDEC_QueryStatus: QueryDecoding channelStatus.

  • CVI_VDEC_SetChnParam: SetDecoding channelParameter.

  • CVI_VDEC_GetChnParam: GetDecoding channelParameter.

  • CVI_VDEC_SendStream: toVideo DecodingChannelSendBitstreamData.

  • CVI_VDEC_GetFrame: GetVideo DecodingChannelDecodingImage.

  • CVI_VDEC_ReleaseFrame: ReleaseVideo DecodingChannelDecodingImage.

  • CVI_VDEC_SetModParam: SetDecodingrelated moduleParameter.

  • CVI_VDEC_GetModParam: GetDecodingrelated moduleParameter.

  • CVI_VDEC_AttachVbPool: willDecoding channelbind to a video buffer VB poolin.

  • CVI_VDEC_DetachVbPool: willDecoding channelunbind from a video buffer VB unbind from the pool.
