8.4. Data Types¶

Video DecodingRelatedData Types, The data structure is defined as follows:

  • VDEC_MAX_CHN_NUM: DecodingMaximumChannelnumber

  • JPEGD_ALIGN_W: jpeg DecodingAlignmentWidth

  • JPEGD_ALIGN_H: jpeg DecodingAlignmentHeight

  • JPEGD_MAX_WIDTH: jpeg DecodingMaximumWidth

  • JPEGD_MAX_HEIGHT: jpeg DecodingMaximumHeight.

  • VDEC_CHN_ATTR_S: Decoding channelAttribute.

  • VDEC_ATTR_VIDEO_S: Video DecodingVideoChannel attributes.

  • VIDEO_MODE_E: BitstreamSendMethodEnumeration.

  • VDEC_CHN_STATUS_S: ChannelStatusStructure.

  • VDEC_DECODE_ERROR_S: DecodingErrorInformationStructure.

  • VDEC_CHN_PARAM_S: Decoding channeladvancedParameterStructure.

  • VDEC_PARAM_VIDEO_S: Video DecodingadvancedParameterStructure.

  • VDEC_PARAM_PICTURE_S: imageDecodingadvancedParameterStructure.

  • VIDEO_OUTPUT_ORDER_E: Decodingoutput orderEnumeration.

  • COMPRESS_MODE_E: DecodingImageCompressionModeEnumeration.

  • VDEC_STREAM_S: DecodingBitstreamStructure.

  • VDEC_USERDATA_S: UserDataStructure.

  • VIDEO_DISPLAY_MODE_E: DisplayModeEnumeration.

  • VDEC_CHN_POOL_S: DefinitionDecoding channelBind VB poolStructure.
