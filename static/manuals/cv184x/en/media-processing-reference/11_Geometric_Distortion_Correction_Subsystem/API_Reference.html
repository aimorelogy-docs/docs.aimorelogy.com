11.3. API Reference¶

This function module provides the following APIs to users:

  • CVI_GDC_Init: GDCsystemInitialize.

  • CVI_GDC_DeInit: GDCsystemDeinitialize.

  • CVI_GDC_BeginJob: startaJob.

  • CVI_GDC_EndJob: endajob,thehasinthisjobinTaskwillbesubmittoGDC Module.

  • CVI_GDC_CancelJob: cancelajob,thehasinthisjobinTasknotwillbesubmittoGDC Module.

  • CVI_GDC_SetJobIdentity: SetJobIdentityAttribute.

  • CVI_GDC_AddRotationTask: inbothhasJobinaRotationTask, Not supportedRotation180.

  • CVI_GDC_GenLDCMesh: isLDCAttributegeneratemesh, meshused to storagerawImageandcorrectionafterImagepixelspointforshouldness.

  • CVI_GDC_AddLDCTask: inbothhasJobinalens correctionTask.

  • CVI_GDC_LoadLDCMesh: UserModeloadLDC mesh.

  • CVI_GDC_DumpMesh: dumpUsemesh.

  • CVI_GDC_LoadMesh: Load the mesh to use.

  • CVI_GDC_LoadMeshWithBuf: Load the mesh to use.

  • CVI_GDC_GetWorkJob: Getcurrentlyinperform Job.

  • CVI_GDC_GetChnFrame: GetChannelFrame.

  • CVI_GDC_Suspend: Suspend GDC.

  • CVI_GDC_Resume: Wake up GDC.
