3. API Reference¶

CIPHER providewithunder APIs:

  • CVI_UNF_CIPHER_Init :initialization CIPHER module.

  • CVI_UNF_CIPHER_Deinit :deinitialization CIPHER module.

  • CVI_UNF_CIPHER_Open: Open the CIPHER module.

  • CVI_UNF_CIPHER_Close :Disable CIPHER module.

  • CVI_UNF_CIPHER_CreateHandle: Create a Cipher handle.

  • CVI_UNF_CIPHER_DestroyHandle :destroyalreadystorein CIPHER handle.

  • CVI_UNF_CIPHER_ConfigHandle :Configure CIPHER control information.

  • CVI_UNF_CIPHER_ConfigHandleEx :Configure CIPHER control information (extended) .

  • CVI_UNF_CIPHER_GetHandleConfig :Get CIPHERconfigureinformation.

  • CVI_UNF_CIPHER_Encrypt :single-packet data encryption.

  • CVI_UNF_CIPHER_Decrypt :single-packet data decryption.

  • CVI_UNF_CIPHER_EncryptVir :fordataperform encryption.

  • CVI_UNF_CIPHER_DecryptVir :fordataperform decryption.

  • CVI_UNF_CIPHER_EncryptMulti :multi-packet data encryption.

  • CVI_UNF_CIPHER_DecryptMulti :multi-packet data decryption.

  • CVI_UNF_CIPHER_HashInit :HASH, HMAC calculateinitializationfunction.

  • CVI_UNF_CIPHER_HashUpdate :HASH, HMAC calculatedatainputfunction.

  • CVI_UNF_CIPHER_HashFinal :HASH, HMAC calculatemostendresultoutputfunction.

  • CVI_UNF_CIPHER_GetRandomNumber :random number retrieval.

  • CVI_UNF_CIPHER_GetTag :TAG value retrieval.

  • CVI_UNF_CIPHER_RsaPublicEncrypt :usepublic keyforplaintextperform encryption.

  • CVI_UNF_CIPHER_RsaPrivateDecrypt :useprivate keyforciphertextperform decryption.

  • CVI_UNF_CIPHER_RsaPublicDecrypt :usepublic keyforciphertextperform decryption.

  • CVI_UNF_CIPHER_RsaSign :useprivate keyforuserdataperform signature.

  • CVI_UNF_CIPHER_RsaVerify :usepublic keyforuserdataperform validnessandcompletenessverification.

  • CVI_UNF_CIPHER_KladEncryptKey :use KLAD forclear keyperform encryption.
